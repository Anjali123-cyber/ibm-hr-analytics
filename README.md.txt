🤖 AI Tool Adoption in Education - Survey Dashboard

A Power BI dashboard created to analyze student and faculty trends in using AI tools for academic tasks. This project visualizes awareness, trust, impact, and willingness to pay across multiple demographics.

🔧 Tools & Technologies
- Power BI
- DAX
- Power Query

🚀 Features
- Awareness & trust levels by year, stream, and state
- KPIs: Willingness to Pay, Usage %, Professor Allowance
- Visualizations: Donut charts, line graphs, stacked bars
- Filters for stream, year of study, device, and internet access

📁 Dataset Source
Primary data collected through survey (CSV available on request)
